Privacy Policy for Fashion Trend Analyzer

Last Updated: July 3, 2025

Introduction

This Privacy Policy describes how Fashion Trend Analyzer ("the App," "we," "us," or "our") handles information. This app is a personal, non-commercial project designed for educational and data analysis purposes. It interacts with the Pinterest API to analyze public trends.

1. Information We Collect

Our App is designed to analyze trends, not people. We do not collect, store, or process any Personally Identifiable Information (PII) from our users or from Pinterest users.

The only data the App handles is aggregated, anonymized trend data provided by the Pinterest API, such as:

  • Trending keywords (e.g., "summer dress," "streetwear").
  • Anonymized search volume and performance metrics for those keywords.

2. How We Use Information

The anonymized trend data is used exclusively for the following purposes:

  • To display fashion trends within the App's dashboard.
  • To analyze changes in trend popularity over time for educational purposes.
  • To learn about data visualization and API integration.

3. Information Sharing and Disclosure

We do not share, sell, rent, or trade any data with third parties. The App is for personal use only and does not have any commercial function. All data retrieved from the Pinterest API is used solely within the App's dashboard for the owner's analysis.

4. Data Security

We take reasonable measures to protect the information within our control. The connection to the Pinterest API is made securely using credentials stored in a safe environment. Since we do not collect any personal data, the risk to individual privacy is minimal.

5. Your Rights

As we do not collect any personal data, there is no personal information to access, change, or delete.

6. Changes to This Privacy Policy

We may update this Privacy Policy from time to time. We will notify you of any changes by posting the new Privacy Policy on this page.

7. Contact Us

If you have any questions about this Privacy Policy, you can contact the project owner through the GitHub repository where this project is hosted.
